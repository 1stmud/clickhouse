# GORM MySQL Driver

## Quick Start

```go
import (
  "github.com/ClickHouse/clickhouse-go"
  "gorm.io/gorm"
)

// https://github.com/go-sql-driver/mysql
dsn := "tcp://localhost:9910?debug=true"
db, err := gorm.Open(sql.Open(dsn), &gorm.Config{})
```
